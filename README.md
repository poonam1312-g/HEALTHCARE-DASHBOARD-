# HEALTHCARE-DASHBOARD-
Project Overview

The Healthcare Analytics Dashboard is a data visualization project that analyzes hospital and patient data to generate meaningful insights about healthcare operations. The dashboard helps understand patient demographics, treatment types, admission trends, department performance, and treatment costs.

The goal of this project is to transform raw healthcare data into clear and interactive visualizations that support better decision-making in healthcare management.

Project Description

This dashboard provides a detailed analysis of healthcare data by examining patient gender distribution, treatment types, insurance providers, admission categories, and medical diagnoses. It also analyzes department performance and the length of hospital stays.

The dashboard highlights the overall treatment cost and allows users to filter data by department to explore trends more effectively.

Tech Stack

Power BI – Data visualization and dashboard creation

Microsoft Excel – Data cleaning and dataset preparation

Power Query – Data transformation

GitHub – Project documentation and version control

Key Insights

Medication and observation are among the most common treatment types.

Patient distribution across departments is relatively balanced, with pediatrics and orthopedics having slightly higher counts.

Emergency admissions represent a large portion of total admissions.

Insurance coverage varies across patients and departments.

The dashboard helps identify diagnosis patterns and hospital resource utilization.

Conclusion
The Healthcare Analytics Dashboard provides valuable insights into patient data, treatment patterns, and hospital operations. By visualizing key healthcare metrics, the dashboard supports data-driven decision making and helps improve healthcare service management.
